# AI Emotion Tracker

A real-time facial emotion detection system using Python, OpenCV, and DeepFace.  
The system analyzes webcam input and displays emotions with confidence levels and visual bars.

## Features
- Real-time emotion detection using webcam
- Detects emotions: angry, happy, sad, surprise, neutral
- HUD-style interface with emotion bars
- Optimized analysis for smooth performance

## Tech Stack
- Python
- OpenCV
- DeepFace
- TensorFlow

## How to Run

1. Clone the repository
2. Install dependencies

pip install -r requirements.txt

3. Run the program

python emotion_tracker.py
